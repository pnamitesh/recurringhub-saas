# RecurringHub SaaS

Recurring billing management system for tuition centers, gyms, and studios.

## Tech Stack
- React 18
- React Router
- Context API
- Chart.js

## Setup
\`\`\`bash
npm install
npm start
\`\`\`

## Deploy
\`\`\`bash
npm run build
vercel --prod
\`\`\`
